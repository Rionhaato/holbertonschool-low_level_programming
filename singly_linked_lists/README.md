# singly_linked_lists

Intro project for singly linked lists in C.

## Files
- 0-print_list.c: print all elements of a `list_t` list.
- lists.h: type definitions and prototypes.
- 0-main.c: provided test harness.

## Build
Example (with provided main):
```
gcc -Wall -Wextra -Werror -pedantic -std=gnu89 0-main.c 0-print_list.c -o a
```

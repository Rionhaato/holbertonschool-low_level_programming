# Doubly Linked Lists

This directory implements basic operations for a doubly linked list of integers.

## Files
- 0-print_dlistint.c: prints all elements of a `dlistint_t` list and returns the node count.
- 0-main.c: sample usage that builds a two-node list and calls `print_dlistint`.
- lists.h: type definition for `dlistint_t` and prototypes for list-related functions.

## Build
Compile with:
```sh
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c 0-print_dlistint.c -o a
```

## Run
```sh
./a
```

# C - malloc, free

This project covers dynamic memory allocation in C using `malloc` and `free`.

## Tasks

- 0. Float like a butterfly, sting like a bee
  - File: `0-create_array.c`
  - Function: `char *create_array(unsigned int size, char c);`
  - Creates an array of chars and initializes it with a specific char.

- 1. The woman who has no imagination has no wings
  - File: `1-strdup.c`
  - Function: `char *_strdup(char *str);`
  - Returns a pointer to a newly allocated space containing a copy of a string.

## Compilation

Use:

```
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 <test>.c <task>.c -o out
```

## Allowed functions

`malloc`, `free`, standard I/O for tests.
